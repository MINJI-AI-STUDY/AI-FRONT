# 학교 가입 및 승인 화면

## 화면
- `/signup/teacher`: 교직원 가입 요청
- `/signup/student`: 학생 가입 요청
- `/operator/signup-requests`: 학교 운영자 가입 승인

## 교직원 가입 요청
- 학교 선택
- 이름
- 로그인 ID
- 비밀번호
- 학교 이메일
- 이용약관/개인정보/학생·학교 고지 동의

## 학생 가입 요청
- 학교 선택
- 실명
- 가입 요청 접수 후 승인 대기 안내 문구 표시
- 이용약관/개인정보/학생 서비스 고지 동의

## 운영자 승인 화면
- 학교 선택
- 대기 중인 가입 요청 목록
- 승인/반려 버튼
- 반려 시 사유 입력 가능
- 승인 시 provisioned 로그인 정보 확인 가능

## 운영자 범위
- 이번 MVP에서 운영자 화면은 학교 단위 가입 승인과 학교 요약 대시보드에 집중합니다.
- 학교/학급/사용자 전반 CRUD는 운영 화면 기본 범위에 포함하지 않습니다.
