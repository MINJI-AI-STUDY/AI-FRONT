# Teacher Workspace 상세 화면 명세

## 화면 경로
- 공식 메인: `/teacher/channels/:channelId/workspace`
- 보조 생성/검토 화면: `/teacher/materials/:materialId/generate`, `/teacher/question-sets/:questionSetId/review`

## 목적
- 교사가 채널에 귀속된 PDF를 보면서 문제 생성, 최근 생성 세트 검토, 배포 상태 확인을 수행한다.
- 교사의 공식 출제 단위는 채널이며, material/question set은 채널 문맥 안에서 다뤄진다.

## 주요 영역
| 영역 | 설명 |
| --- | --- |
| 상단 header | 채널 제목, 역할 메타, 교사 홈 복귀 |
| 좌측 channel list | 교사 채널 목록 |
| 중앙 문서 canvas | 현재 채널에 연결된 PDF inline viewer |
| 우측 문제 흐름 패널 | 문제 생성 시작, 최근 생성 세트 검토, 배포 상태/코드 확인 |
| 보조 생성/검토 화면 | 실제 generate/review 작업을 이어가는 페이지 |

## 상태
| 상태 | 설명 |
| --- | --- |
| loading | 자료 로딩 중 |
| error | 자료 조회 실패 |
| idle | 생성 전 |
| generating | 생성 요청 중 |
| generated | JSON 결과 표시 |
| published | 배포 코드 / 마감 상태 확인 |

## 액션
- 문제 생성
- 검토 화면 열기
- 최근 배포 세트 보기

## UX 원칙
- PDF가 항상 중앙 주인공이어야 한다.
- 문제 생성/검토/배포는 채널 문맥에서 시작해야 한다.
- material standalone 페이지는 보조 작업 화면이며, 공식 메인 학습/출제 단위는 채널이다.
