# Student Workspace 상세 화면 명세

## 화면 경로
- 공식 메인: `/student/channels/:channelId/workspace`
- 집중 풀이: `/student/question-sets/:distributionCode/workspace`

## 목적
- 학생이 같은 PDF를 보면서 채널 대화와 자료 기반 AI 질의응답을 오가고,
- 활성 문제 세트가 있으면 같은 채널 문맥에서 바로 문제 풀이로 진입한다.

## 주요 영역
| 영역 | 설명 |
| --- | --- |
| 상단 header | 채널 제목, 역할 메타, 학생 홈 복귀 |
| 좌측 channel list | 참여 채널 목록, 전체 패널 단위 collapse |
| 중앙 문서 canvas | 현재 채널에 연결된 PDF inline viewer |
| 우측 mode panel | `AI 도우미` / `채널 대화` 전환형 패널 |
| 문제 풀이 entry | 활성 question set이 있으면 PDF 문맥 안에서 즉시 진입 |
| 모달 | 집중 문제 풀이 모달 |

## 상태
| 상태 | 설명 |
| --- | --- |
| loading | 문제 세트 로딩 |
| error | 문제 세트 또는 QA 요청 실패 |
| noActiveSet | 현재 자료에 연결된 배포 문제 없음 |
| answering | QA 요청 중 |
| submitting | 제출 중 |
| responded | QA 응답 표시 |

## UX 원칙
- 문서 맥락을 끊지 않는다.
- 문제풀이와 QA는 같은 자료 컨텍스트를 공유한다.
- AI와 채널 대화는 우측 사이드바 안에서 전환되며, 중앙 PDF가 항상 주인공이어야 한다.
- distribution code는 진입 수단일 뿐, 학생의 기본 학습 단위는 채널이다.
