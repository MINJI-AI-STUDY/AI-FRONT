# 화면 카탈로그

| 화면명 | 경로 | 설명 |
| --- | --- | --- |
| 로그인 | `/login` | 역할별 데모 로그인 |
| 교사 홈 | `/teacher` | 교사 진입 화면 |
| 학생 홈 | `/student` | 학생 진입 화면 |
| 교사 채널 workspace | `/teacher/channels/:channelId` | 교사 채널 운영, PDF, roster, 메시지 |
| 학생 채널 workspace | `/student/channels/:channelId` | 학생 채널 PDF, 메시지, 서브 챗봇 |
| 교사 workspace | `/teacher/materials/:materialId/workspace` | teacher 공통 문서 workspace |
| 학생 workspace | `/student/question-sets/:distributionCode/workspace` | student 공통 문서 workspace |
| 운영자 overview | `/operator` | 운영자 요약 대시보드 |
