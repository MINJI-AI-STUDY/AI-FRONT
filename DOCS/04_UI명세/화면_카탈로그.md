# 화면 카탈로그

| 화면명 | 경로 | 설명 |
| --- | --- | --- |
| 로그인 | `/login` | 교사/학생/학교 운영자 일반 로그인 |
| 교직원 가입 요청 | `/signup/teacher` | 학교 이메일 기반 교직원 가입 요청 |
| 학생 가입 요청 | `/signup/student` | 실명 기반 학생 가입 요청 |
| 가입 승인 | `/operator/signup-requests` | 학교 운영자 가입 승인/반려 |
| 교사 홈 | `/teacher` | 교사 진입 화면, 첫 채널 생성 CTA, 문서 대시보드 모달 진입 |
| 학생 홈 | `/student` | 학생 진입 화면, 제출 결과 모달 진입 |
| 교사 채널 workspace | `/teacher/channels/:channelId` | 교사 채널 운영, PDF, roster, 메시지 |
| 학생 채널 workspace | `/student/channels/:channelId` | 학생 채널 PDF, 메시지, 서브 챗봇 |
| 교사 자료 상태 | `/teacher/materials/:materialId` | 업로드 상태, 대시보드, workspace 진입 |
| 교사 문서 대시보드 | `/teacher/materials/:materialId/dashboard` | 문서 통계, 최근 QA, 생성 문제 세트, 문제세트 대시보드 모달 진입 |
| 교사 문제 검토 | `/teacher/question-sets/:questionSetId/review` | 생성 문항 검토/수정/배포, due date 입력 |
| 교사 문제 대시보드 | `/teacher/question-sets/:questionSetId/dashboard` | 학생 점수/정답률/취약 개념 |
| 교사 workspace | `/teacher/materials/:materialId/workspace` | teacher 공통 문서 workspace |
| 학생 workspace | `/student/question-sets/:distributionCode/workspace` | student 공통 문서 workspace |
| 학생 문제 풀이 | `/student/question-sets/:distributionCode` | 문제 세트 풀이/제출 |
| 학생 제출 결과 | `/student/submissions/:submissionId` | 점수/해설/관련 개념 확인 |
| 학생 QA | `/student/materials/:materialId/qa` | 문서 기반 질문/이력 |
| 운영자 overview | `/operator` | 학교 운영자 요약 대시보드, 가입 승인 진입 |
